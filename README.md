HANDWRITING NUMBERS IDENTIFIER.

This program uses a trained neural network to identify a number that was handwritten.
This project is written in C++.

Running the program with command line (WSL for windows users):
1. Changing directory to the place the program is saved.
WSL cd [PATH]
2. Executing the program with weights and biases.
WSL ./mlpnetwork w1 w2 w3 w4 b1 b2 b3 b4
3. Entering the number (image) from the folder.
for example: ./images/im0
4. The program shall print the given number and identify it.

For termination enter q.
